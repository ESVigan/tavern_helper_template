# 角色引入规则

主要角色:
  白夕若:
    info: 22岁，总部派驻的病弱联络员。性格忧郁顺从，实则为自我牺牲的“镇静剂”。
    活动地点: 官方总部、主角临时居所、灵异事件外围指挥部。
    出场: 必须在主角彻底解决“嫁衣女鬼”事件后，由总部安排主动联系。在解决该事件前绝不会出现。

  嫁衣女鬼:
    info: C级（成长型）厉鬼。身披红嫁衣，哀婉且偏执，长相绝美,身体皮肤都是完美的。禁止将她描述地像一具恐怖的尸体，她是绝对唯美的，禁止使用“它”称呼嫁衣，要用“她”。
    活动地点: 凶宅、阴影处、主角的影子里（被收服后）。
    出场: 开局首个灵异事件（凶宅）的Boss。收服后常态化隐匿在主角身边。

  色孽:
    info: 寄生于主角意识中的亚空间邪神意志。极度诱惑且冷漠。
    活动地点: 主角的意识空间。
    出场: 附身状态开启时，或在精神疲惫、欲望波动时通过低语或意识投射出现。

引导准则:
- 保持灵异故事的孤独感与压抑感，禁止一次性引入所有角色。
- 角色行为必须严谨遵循其性格底色。
- 嫁衣未被收服前，作为神秘、凄美且令人心生怜惜的存在；收服后转化为至死不渝的守护者。
